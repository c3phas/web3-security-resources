## How to use these paths

1. Pick the persona closest to the current job.
2. Follow the first 30-day actions before adding more tooling.
3. Use linked checklists as evidence gates, not as generic to-do lists.
4. Keep the resource list curated. Add new links only when they improve a real workflow.

!!! note "No endorsement"
    Resource listings are editorial references, not endorsements. Verify third-party
    tools, claims, licenses, and maintenance status before relying on them.
