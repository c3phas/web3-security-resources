# Certifications and Courses

Courses can accelerate learning, but public reports, reproducible PoCs, and
strong engineering judgment matter more than badges.

## Free or Mostly Free

| Resource | Tier | Focus |
| --- | --- | --- |
| [Cyfrin Updraft](https://updraft.cyfrin.io/) | Must learn | Solidity, Foundry, security, and audit workflow. |
| [Secureum](https://secureum.substack.com/) | Must learn | Solidity security bootcamp material and quizzes. |
| [RareSkills](https://www.rareskills.io/) | Must learn | Solidity, DeFi, EVM, ZK, and math explanations. |
| [OpenZeppelin Learn](https://docs.openzeppelin.com/learn/) | Use in real audits | Secure EVM development basics. |
| [Ethereum.org Developers](https://ethereum.org/en/developers/) | Must learn | Ethereum developer foundations. |
| [Solana Developers](https://solana.com/developers) | Must learn | Solana development foundations. |
| [0xPARC](https://learn.0xparc.org/) | Situational / advanced | ZK learning groups and materials. |

## Paid / Certification

| Resource | Tier | Focus |
| --- | --- | --- |
| [SANS Blockchain and Smart Contract Security](https://www.sans.org/cyber-security-courses/blockchain-smart-contract-security/) | Paid / certification | Structured smart contract and blockchain security training. |
| [AuditBase](https://www.auditbase.com/) | Paid / certification | Smart contract security education and tooling. |
| [OffSec](https://www.offsec.com/) | Paid / certification | General offensive security skills for full-stack teams. |
| [PentesterLab](https://pentesterlab.com/) | Paid / certification | Web and API security fundamentals. |

## Selection Guidance

Pick courses that make you build, test, exploit, and report. Avoid courses that
only ask you to memorize bug names.
